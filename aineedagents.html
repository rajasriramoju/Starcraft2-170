OVERVIEW

This project is a tool to aid StarCraft II players in with suggestions for their game, using past replays. The tool uses past replays to analyze what the best course of action is, in terms of number of units a player should have at a specific time interval in the game. The tool can currently used on a map where the match is held between Protoss v/s Protoss.
